القوائم والقواميس المتداخلة

والان بعد ما رأينا القوائم في بايثون في الدروس السابقة والقواميس ايضا الان اريد ان اتحدث عن مفهوم مهم متعلق بهم وهو ما يسمى بالتداخل او nesting

فلو تخيلت قائمة او قاموس كمجلد يمكنك وضع اشياء داخله فالتداخل هو ببساطة ان تضع مجلد داخل مجلد اخر او بالاحرى قائمة داخل قائمة او قاموس داخل قاموس او حتى قاموس داخل قائمة وهكذا

هذا هو الشكل العامل للقاموس 


{
    Key: Value,
    Key2: Value2,
}

فيمكننا في مكان ان تكون القيمة للمفتاح قيمة واحدة يمكن ان تكون قائمة وايضا يمكن ان تكون قاموسا ايضا 


{
    Key: [List],
    Key2: {dict},
}

ربما تصبح معقدة قليلا ولكن تتيح لنا العديد من الامكانيات لتخزين المعلومات

الان بعض الامثلة الواقعية:

مثلا قاموس لتخزين سجل الاسفار بحيث المفاتيح هي الدول التي زرتها والقيم هي الاماكن التي زرتها في ذلك البلد 


travel_log = {
    "France": ["Paris", "Lille", "Dijon"],
    "Germany": ["Berlin", "Hamburg", "Stuttgart"],
}

تذكر دائما ان المفتاح في القاموس يأخذ قيمة واحدة فلو اردت ان تعطيها اكثر من قيمة نستعمل قائمة

الان نريد ان نقوم بوضع قاموس داخل قاموس فلو نعدل على القاموس السابق travel_log بحيث نبين ان قائمة المدن هذه هي قائمة بالمدن التي زرتها لانها ليست واضحة تماما
حاول فيها كتحدي 


travel_log = {
    "France": {
        "cities_visited": ["Paris", "Lille", "Dijon"]
    },
    "Germany": {
        "cities_visited": ["Berlin", "Hamburg", "Stuttgart"]
    },
}

الان يمكنني ان اضيف ثنائمة جديدة للقاموس الخاص بالدول يسمى "total_visits" اخزن فيه عدد مرات الزيارة لكل بلد 


travel_log = {
    "France": {
        "cities_visited": ["Paris", "Lille", "Dijon"],
        "total_visits": 10,
    },
    "Germany": {
        "cities_visited": ["Berlin", "Hamburg", "Stuttgart"],
        "total_visits": 12,
    },
}

حاول صناعة travel log خاص بك كتحدي لك تضيف فيه كل الاماكن التي زرتها وعدد المرات

الان بعد ما رأينا قائمة داخل قاموس وقاموس داخل قاموس الان سوف نرى قواميس داخل قائمة 


[{
    Key: [list],
    Key2: {Dict},
},
{
    Key: Value,
    Key2: Value2,
}]

بمعنى قائمة من القواميس و القاموس الاول عند المؤشر 0 والثاني عند 1 وهكذا كقائمة عادية بينما في القاموس القيم تحدد بالمفاتيح مثلا في المثال السابق "France" او "Germany"

سوف احول travel_log الى قائمة قواميس بحيث كل قاموس فيه الدولة كمفتاح "country" والاماكن التي زرتها "cities_visited" وعدد مرات الزيارة total_visits 


travel_log = [
    {
        "country": "France",
        "cities_visited": ["Paris", "Lille", "Dijon"],
        "total_visits": 10,
    },
    {
        "country": "Germany",
        "cities_visited": ["Berlin", "Hamburg", "Stuttgart"],
        "total_visits": 12,
    },
]

وهكذا لدينا قائمة فيها عنصرين وهما عبارة عن قاموسين وكل قاموس فيه 3 ثنائيات الاولى قيمتها عبارة عن string والثانية قيمتها عبارة عن قائمة والثنائية الثالثة قيمتها عبارة عن عدد int

في الدرس القادم سوف تختبر في الاشياء التي تعلمتها في درس اليوم.
