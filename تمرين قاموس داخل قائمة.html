تمرين قاموس داخل قائمة

التعليمات:

سوف نقوم بكتابة برنامج يقوم بالاضافة الى ال travel_log وهو عبارة عن قائمة تحتوي على قاموسين من الدرس السابق 


travel_log = [
    {
        "country": "France",
        "cities_visited": ["Paris", "Lille", "Dijon"],
        "total_visits": 10,
    },
    {
        "country": "Germany",
        "cities_visited": ["Berlin", "Hamburg", "Stuttgart",
        "total_visits": 12,
    },
]

اكتب دالة التي تتيح لك اضافة زيارات الى ال travel_log مثلا لو زرت روسيا والامامكن التي زرتها هي Moscow, Saint Petersburg وعدد مرات الزيارة هو 2 تقوم بمنادات الدالة هكذا: 


add_new_country("Russia", 2, ["Moscow", "Saint Petersburg"])

وعندما تقوم بطباعة ال travel_log 


travel_log = [
    {
        "country": "France",

    {
        "country": "Russia",
        "cities_visited": ["Moscow", "Saint Petersburg"],
        "total_visits": 2,
    },
]

حظا موفقا في حل التمرين.

حل التمرين: 


def add_new_country(country_visited, times_visited, cities_visited):
    new_country = {}
    new_country["country"] = country_visited
    new_country["cities_visited"] = cities_visited
    new_country["total_visits"] = times_visited
    travel_log.append(new_country)

عرفنا دالة جديدة تسمى add_new_country تأخد 3 مدخلات: الدولة وعدد الزيارات والاماكن التي زرتها وماتفعله هذه الدالة هو انها تحول هذه المدخلات الى قاموس ثم تضيفها الى قائمة ال travel_log
